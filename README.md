# url-shrinker

### simple and powerful link management tool



## Getting started

Install JSON Server 

```
npm install
```
run the backend (shrinker APIs)
```
node server.js
```
now you can acces all the RESTful APIs 

## getting started with the react(client) app

install all the dependencies
```
npm install
```
run the client app
```
npm start
```
